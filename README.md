## AI Writing Assitant 

This is a Ai powered writing Assistant used to help you in daily writing task like writing blogs, Linkedin posts , Stories etc
This uses Gemini .


## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
